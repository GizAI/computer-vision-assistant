# Autobot UI

This directory contains the Vue 3 frontend for Autobot.

## Structure

- `src/`: Vue source code
- `public/`: Static assets
- `dist/`: Built frontend (generated)

## Development

1. Install dependencies:
   ```
   npm install
   ```

2. Start development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

## Technologies

- Vue 3
- shadcn-vue
- TypeScript
- Vite
